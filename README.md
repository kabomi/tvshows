# TV Shows App
This application allows users to look for TV shows based on different genres (drama, comedy, sports, etc.).
TV shows data is served by the following open API: http://www.tvmaze.com/api

## Prerequisites
- Node Version >= 18.18.2
- Npm Version >= 9.8.1
